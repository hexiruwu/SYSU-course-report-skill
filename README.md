# 课程报告生成 Skill

> 适用于中山大学智能工程学院党政扬同学的课程作业报告自动生成工具，基于 [Claude Cowork](https://claude.ai) Skill 系统。

---

## 功能简介

每次只需告诉 Claude "帮我写这个作业"，Skill 自动处理：

- ✅ 双栏学术期刊模板（`article` + `multicols`）或单栏 SYSUReport 模板，根据作业类型自动选择
- ✅ 完整 LaTeX 源码，可直接在 Overleaf 编译
- ✅ 规范的图片占位 + 三层注释结构（没有图片也能编译，准备好图片后一键替换）
- ✅ AI 生图 Prompt / 文献来源链接 / 用户截图指引，三类图片统一规范
- ✅ 数学公式（带编号）、数据表格、代码块等工科报告标准元素
- ✅ 参考文献自动按领域新旧程度筛选（热门新工科优先引用近 3 年文献）
- ✅ 实验类报告体现过程思考，论文类报告侧重理论深度
- ✅ 生成图片清单 + 参考文献建议，方便补充

---

## 快速开始

### 安装

1. 下载 [Releases](../../releases/latest) 中的 `course-report.skill` 文件
2. 在 Claude 桌面端 → Settings → Capabilities → Skills，点击安装，选择该文件

### 使用

直接在 Claude 对话中描述作业要求即可触发，例如：

```
老师发了作业要求（粘贴或上传文件）：...
帮我写这个作业报告，主题是 XXX
```

```
计算机视觉课要写一篇关于 ViT 的期末报告，3000 字以上，要有公式推导
```

```
强化学习作业，主题是 PPO 机器人控制，老师没规定格式，2500 字
```

## 模板说明

### 模板 A：双栏学术期刊（论文/调研类）

- 中山大学学报风格页眉
- `gbt7714-numerical` 参考文献格式
- 双栏正文 + 跨栏大图支持
- 摘要 + 关键词 + 引言贡献列表

### 模板 B：单栏 SYSUReport（实验/编程类）

- 封面页（自动填充姓名、学号、学院、专业）
- 摘要页 + 目录页
- `listings` 代码块（语法高亮）
- `subfigure` 多图对比支持

## 图片使用说明

每张图在 `.tex` 文件中都有以下三层结构：

```latex
% -------------------------------------------------------
% 图1：ViT 整体架构图
% 来源类型：文献图
% 来源：Dosovitskiy et al., 2020, An Image is Worth 16x16 Words, 原文图1
% 下载链接：https://arxiv.org/abs/2010.11929
% -------------------------------------------------------

% === 真实插图（准备好图片后取消注释，删除下方占位）===
%\begin{figure*}[t]
%    ...
%\end{figure*}

% === 占位（插入真实图片后删除此块）===
\begin{figure*}[t]
    \centering
    \fbox{\parbox{...}{图1占位：ViT整体架构图}}
    ...
\end{figure*}
```

**操作步骤**：
1. 按注释中的指引下载/生成/截取图片，放入 `fig/` 目录
2. 删掉占位块
3. 取消注释真实插图代码
